Analyze AirBnB listings in Paris to determine the impact of recent regulations

The situation is You've been hired as a Performance Analyst for AirBnB, a platform that allows individuals to rent out their homes for travellers.
You've been asked to analyze Paris listings, with a focus on pricing. Leadership wants a visual summary of factors affecting pricing and whether regulations adopted in 2015 impacted listings in the Paris Market

- 1. Explore and profile the data to correct any quality issues
- 2. Prepare and reformat the data for visualization
- 3. Visualize the data and identify key insights and recommendations

This project was done in Python, using Jupyter Notebook

Dataset for this project were imported from [Maven-datasets](https://maven-datasets.s3.amazonaws.com/Airbnb/Airbnb+Data.zip)
