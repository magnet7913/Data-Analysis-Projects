The Dataset belongs to MavenToys, a toy store chain with mutiple store locations in Mexico. The goal of this dashboard is to be a simple, interactive report that the leadership team can use to monitor key business metrics and high-level trends!

The Dataset was downloaded from [Maven Analytics](https://maven-datasets.s3.amazonaws.com/Maven+Toys/Maven+Toys.zip)
