A dashboard showcasing important factors for all available LEGO set in the USA, including the set theme, age range, number of pieces and price. This dashboard was aimed to be an interactive tool that you can use to quickly explore all LEGO sets and narrow down the options based on specific criteria!

The Dataset was downloaded from [Maven Analytics](https://maven-datasets.s3.amazonaws.com/LEGO+Sets/LEGO+Sets.zip)
